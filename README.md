# Venkatesh T — Portfolio

Personal portfolio for Venkatesh T, a Data Analytics and Engineering Apprentice at Fidelity Business Services India.

**Live site:** [venkatesh-portfolio-amber.vercel.app](https://venkatesh-portfolio-amber.vercel.app/)

## Highlights

- Data engineering experience with Snowflake, SQL, Python, Tableau, and data governance
- Featured projects including MedTrack, IPL Strategy Engine, and TASS
- Responsive project gallery with image zoom, contact form, resume link, and GitHub/LinkedIn links

## Tech Stack

Next.js, React, Tailwind CSS, Framer Motion, and React Icons.

## Run Locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
